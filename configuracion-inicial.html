Configuración para familia

Cómo dejar Ezivia lista para uso diario

Esta guía resume lo que sí hace la app hoy: configurar Ezivia como launcher, ajustar favoritos y usar llamadas, videollamadas y SOS de forma directa.

Configuración inicial

El flujo principal permite activar Ezivia como launcher y dejar visibles las acciones más usadas.

  • Activar Ezivia como pantalla de inicio del teléfono.
  • Revisar permisos básicos para llamadas, contactos y cámara según necesidad.
  • Configurar contactos favoritos para acceso rápido desde la pantalla principal.

Acceso SOS

Ezivia muestra un contacto SOS al 112 en la pantalla principal para emergencias.

  • El contacto SOS aparece de forma visible junto a favoritos.
  • El acceso está pensado para actuar rápido en caso de urgencia.
  • No depende de menús complejos para llegar a la acción principal.

Control de uso

La app prioriza un entorno de uso simple con menos riesgo de cambios involuntarios.

  • Acceso a ajustes restringido por pulsación larga.
  • Opciones para gestionar favoritos visibles en la pantalla principal.
  • Salida del modo launcher disponible desde ajustes restringidos.

Operación diaria

Funciones disponibles en la versión actual para comunicación y tareas rápidas.

  • Llamadas de voz desde contactos favoritos.
  • Videollamada por WhatsApp con alternativa nativa del sistema.
  • Acciones rápidas de fotos y linterna desde la pantalla principal.
