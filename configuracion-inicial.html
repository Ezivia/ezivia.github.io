Cómo dejar Ezivia lista para uso diario

Esta guia resume lo que si hace hoy la aplicacion: poner Ezivia como pantalla principal, ajustar favoritos y usar llamadas, videollamadas y SOS de forma directa.

Configuración inicial

Con unos pocos pasos puedes dejar Ezivia como pantalla principal y mostrar lo mas importante.

  • Activar Ezivia como pantalla de inicio del telefono.
  • Revisar permisos basicos para llamadas, contactos y camara segun necesidad.
  • Configurar contactos favoritos para tenerlos a mano en la pantalla principal.

Acceso SOS

Ezivia muestra el boton SOS 112 en la pantalla principal para emergencias.

  • El contacto SOS aparece de forma visible junto a favoritos.
  • El acceso esta pensado para actuar rapido en caso de urgencia.
  • No depende de menus complejos para llegar a la accion principal.

Control de uso

La aplicacion prioriza un uso simple con menos riesgo de cambios involuntarios.

  • Acceso a ajustes protegido al mantener pulsado.
  • Opciones para gestionar favoritos visibles en la pantalla principal.
  • Salida del modo sencillo disponible desde ajustes protegidos.

Operación diaria

Funciones disponibles en la version actual para comunicacion y tareas rapidas.

  • Llamadas de voz desde contactos favoritos.
  • Videollamada por WhatsApp con opcion del propio telefono cuando haga falta.
  • Acciones rápidas de fotos y linterna desde la pantalla principal.
